
# Sistema Consulta ANVISA (Streamlit)

Aplicação simples para inserir automaticamente a coluna:
'Nº REGISTRO NA ANVISA' em planilhas Excel.

## Como usar

1. Suba estes arquivos no GitHub
2. Conecte ao Streamlit Cloud
3. Defina o arquivo principal:

app_streamlit_anvisa.py

## Arquivos do projeto

- app_streamlit_anvisa.py
- requirements.txt
- .streamlit/config.toml
